# Naribandhu – Your Health Companion

## 🌸 About the Project
Naribandhu is a simple and accessible menstrual health assistant designed to support menstrual health awareness, especially in rural areas. It helps users track their cycle, predict future periods, and understand possible PCOS risk in an easy and user friendly way.

## 🚀 Features

### 📅 Cycle Tracker
- Predict next period date
- Estimate ovulation day
- Show fertile window

### 🌸 PCOS Risk Checker
- Symptom-based risk assessment
- Categorizes risk into:
  - 🟢 Low Risk  
  - 🟠 Moderate Risk  
  - 🔴 High Risk  
- Provides personalized health tips

### 🎯 Clean UI & UX
- Soft, user-friendly design
- Card-based menu interface
- Smooth animations & visual feedback


## 🛠️ Tech Stack
- Python
- Flask
- HTML
- CSS

## 🎯 Goal
To build a lightweight, beginner-friendly health assistant that promotes menstrual health awareness and early understanding of PCOS.

## 📌 Future Improvements
- 🔊 Voice-based assistance  
- 🌐 Multi-language support (for rural accessibility)  
- 📊 Smarter predictions using past cycle data  
- 📱 Mobile-responsive UI  
- 🔔 Reminder system  

---
## ⚠️ Disclaimer
This application provides general predictions based on standard cycle assumptions and is intended for **educational and awareness purposes only**.  
It should not be used as a substitute for professional medical advice.


## ▶️ How to Run
1. Install Python
2. Install Flask:
   py -m pip install flask
3. Run the app:
   python app.py
4. Open in Browser
   http://127.0.0.1:5000/
