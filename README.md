# Naribandhu – Your Health Companion

## 🌸 About the Project
Naribandhu is a simple and accessible menstrual health assistant designed to support women, especially in rural areas. It provides basic health insights while maintaining privacy and ease of use.

## 🚀 Features
- Period prediction system
- Clean and simple user interface
- Beginner-friendly and lightweight design
- (Upcoming) PCOS risk awareness feature

## 🛠️ Tech Stack
- Python
- Flask
- HTML
- CSS

## 🎯 Goal
To build a low-cost, easy-to-use health assistant that can help young girls and women track their menstrual health and become more aware of conditions like PCOS.

## 📌 Future Improvements
- PCOS risk prediction system
- Voice-based assistance
- Mobile-friendly UI
- Multi-language support for rural accessibility

## ▶️ How to Run
1. Install Python
2. Install Flask:
3.   py -m pip install flask
4. Run the app
5.   python app.py
6. Open in Browser
7.   http://127.0.0.1:5000/
